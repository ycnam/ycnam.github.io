N screen, Cloud가 필요없는 사람들
Monday, Jun 4, 2012
Tags: UX userModel Mobile Nscreen cloud

N screen이나 Cloud 이야기가 많이 나왔던 것에 비하여 이 영역에서 크게 성공한 서비스가 많지 않은 것 같다. 멋들어진 사례는 많지만, 정말 대중적인 성공은 얼마나 되는가? N screen의 중요성은 스마트폰이 대중화되면서 대두되었던 것으로 기억한다. 얼마전까지만 해도 '컴퓨터'라는 것은 한 가정에 하나 있을까 말까 한 것이었는데, 이제는 컴퓨터라고 불리는 것들이 너무나 많아졌다. (모든 가정이 그런건 절대 아니지만) 거실에 데스크탑, 각자가 가지고 있는 랩탑, 주머니 속에는 스마트폰, '소파에 앉은' 시간까지 쪼개어 비집고 들어온 태블릿까지.
